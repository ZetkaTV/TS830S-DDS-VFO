## AD9833 based digital VFO for kenwood TS830/530/other radios :)

### Pinout:

| Arduino   | AD9833 | Encoder |
|-----------|--------|---------|
| 5V        | VCC    |         |
| GND       | DGND   |         |
| 11 (MOSI) | SDATA  |         |
| 13 (SCK)  | SCLK   |         |
| 10 (SS)   | FSYNC  |         |
| 2         |        | CLK     |
| 3         |        | DT      |
| 4         |        | SW      |
